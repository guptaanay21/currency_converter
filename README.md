# 💱 Currency Converter (Python CLI Project)

## 📌 Overview
This is a Currency Converter application built using Python.  
It uses a real-time exchange rate API to convert currencies, check exchange rates, and display supported currencies.

---

## 🚀 Features
- 🔄 Convert currency from one to another
- 📊 Get real-time exchange rates
- 📃 View list of available currencies
- ❌ Error handling for invalid inputs
- 🖥️ Simple and interactive CLI interface

---

## 🛠️ Tech Stack
- Python
- Requests Library (API handling)

---

## ⚙️ How It Works
The program fetches live exchange rate data from an API and allows the user to:
- Enter a command (`list`, `convert`, `rate`, `q`)
- Provide required inputs (currency codes, amount)
- Get instant conversion results
